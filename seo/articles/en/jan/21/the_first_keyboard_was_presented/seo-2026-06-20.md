**FAQ Block:**

Q1: When was the first computer keyboard introduced?  
A1: The first computer keyboard was presented on January 21, 1946, alongside the ENIAC computer.

Q2: How did early data input work before keyboards?  
A2: Engineers used switches, plugs, and punch cards to input data into early computers like ENIAC.

Q3: What was significant about the teletype used with ENIAC?  
A3: It transmitted signals directly into the computer's memory and printed results on paper tape, marking a milestone in input/output technology.

Q4: Did ENIAC have a keyboard?  
A4: No, ENIAC initially did not have a keyboard; input was through switches and punch cards. The first keyboard concept appeared later.
