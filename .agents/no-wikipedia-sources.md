description: Never add Wikipedia as a source in article frontmatter
metadata:
  type: feedback
---

Never add Wikipedia URLs as entries in the `sources:` list of any article's frontmatter.

**Why:** Wikipedia is a secondary source and not authoritative enough to cite. Sources should be primary or reputable original references (official sites, papers, press releases, journalist coverage, author's own writing, etc.).

**How to apply:** When preparing or editing an article, skip Wikipedia links entirely when building the `sources:` list. If no better source is available yet, leave `sources:` empty or omit the entry rather than filling it with Wikipedia.
