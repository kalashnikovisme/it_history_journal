**Visual Studio Code** — доминирующая среда разработки (IDE) сегодня. В большинстве опросов программистов о том, какую IDE они используют, VS Code будет занимать первое место с внушительным отрывом.
29 апреля 2015 Microsoft представила Visual Studio Code на конференции Build, preview-версия была опубликована в тот же день. Стабильный релиз Visual Studio Code произошёл 18 ноября 2015 года.
Visual Studio Code распространяется по одной из самых свободных лицензий программного обеспечения **MIT License**. Лицензия MIT позволяет использовать исходный код продукта в своих продуктах, даже закрытых. Это привело к появлению нескольких популярных форков, самый популярный из них — IDE для удобной разработки с использованием LLM, **Cursor**.
Как может показаться из названия Visual Studio Code — это родственник Microsoft Visual Studio, одной из самых успешных IDE в истории. Но нет, Visual Studio Code не имеет внутри себя модулей от Visual Studio, не является "облегчённой" версией Visual Studio, и общего у них только название и нахождение в инфраструктуре Microsoft.
