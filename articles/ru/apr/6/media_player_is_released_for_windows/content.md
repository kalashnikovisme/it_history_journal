**Windows Media Player** — одна из самых легендарных программ в истории. Подавляющее большинство пользователей компьютера свой первый цифровой аудио-файл прослушала именно через эту программу.
В конце 90-х и начале 2000-х Windows Media Player был основным проигрывателем музыки для большинства пользователей компьютера. Надо упомянуть, что неединственным. Для пользователей Windows были доступны разные программы, более продвинутые программы для прослушивания музыки. При том в каждом регионе мира эти программы были свои. В местах, где я рос, главными конкурентами Windows Media Player были Winamp и AIMP.
Но они точно уступали в доступности Windows Media Player, так как он уже был предустановлен на всех компьютерах с Windows.

## Windows 3.1

Первый релиз этой программы случился 6 апреля 1992 года вместе с **Windows 3.1**. Тогда она просто называлась Media Player. Название Windows Media Player стало официальным уже 1996 году.

Windows 3.1 содержал эти нововведения:

- система TrueType-шрифтов (конкуренция Adobe Type Manager);
- улучшенная мультимедийная поддержка (звук, MIDI, CD-audio);
- появление Media Player, Sound Recorder и экранных заставок;
- улучшенные File Manager и Control Panel.

Сам Media Player умел не многое:

- воспроизводить MIDI-файлы
- проигрывать CD-Audio
- работать с некоторыми AVI-видео
- использовать MCI (Media Control Interface) для управления мультимедией
- запускать звук через Sound Blaster и другие ранние звуковые карты

Но даже этого было достаточно, чтобы запустить эру мультимедиа на Windows.

## Windows Media Player

Свою максимальную популярность и легендарность Windows Media Player получил в 1996 году. Технически это было уже совсем другое приложение. Windows Media Player умел проигрывать видео разных форматов и потоковое аудио.
Легендарные визуализации Windows Media Player появились ещё позже, в релизе 2000 года.
